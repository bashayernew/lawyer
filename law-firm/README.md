Law Firm (Next.js + Tailwind)

Run locally:
- cd law-firm
- npm install
- npm run dev

Open http://localhost:3000

Env (email) — create .env.local:
- SMTP_HOST=your.smtp.host
- SMTP_PORT=587
- SMTP_USER=your_user
- SMTP_PASS=your_pass
- MAIL_FROM=noreply@example.com
- MAIL_TO=you@example.com

Routes:
- / — homepage
- /about — firm overview
- /team — team listing
- /services — services + request form
- /contact — contact form + map
